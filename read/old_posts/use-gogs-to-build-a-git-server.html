使用 Gogs 搭建一台 Git 服务器
作者: xtlsoft     时间: 2017-08-19 10:34:53     分类: 旧稿

由来

Github 太慢，无意间看到 Gogs，就像搞个镜像。 没想到这么好用。

公共 Git 服务

我搭建了一个公共 Git 服务： https://git.xapps.top/

（整理时补充：已不可使用）

遇到的坑

在重启服务器（不管是 Nginx 还是 Gogs）之后，已经登陆的用户再次访问会出现invalid csrf token错误。 解决：关闭 Nginx 的缓存，清除浏览器 cookies。
